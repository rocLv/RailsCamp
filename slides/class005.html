数组比较

> Comparison — Returns an integer (-1, 0, or +1) if this array is less than,
> equal to, or greater than other_ary.

!ruby
arr = [1,2,3]
another_arr = [1,2,4]

arr <=> another_arr # => -1

arr = [1, 5, 3]
another_arr = [1, 2, 3]

arr <=> another_arr # => 1 (5 is greater than 2)

递归

视频

斐波那契数列的实现 0, 1, 1, 2, 3, 5, 8, 13..., 求第n个数的值

1 def fib(n)
2   return n if n <= 1
3   fib(n-1) + fib(n-2)
4 end

阶乘 fact(n) = n! = 1 x 2 x 3 x ... x (n-1) x n = (n-1)! x n = fact(n-1) x n

1 def fac(n)
2   return 1 if n == 1
3   n * fac(n - 1)
4 end
